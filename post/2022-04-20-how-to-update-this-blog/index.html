How to Update This Blog

Posted by     Zoey on Wednesday, April 20, 2022

This blog is built by Hugo and deployed to GitHub Pages. When updating new contents, I use Git pushing everything to the GitHub repository, and here are some memo to myself:

#1 Write a post

  1. Open cmd, cd to hugo blog local folder.
hugo new posts/“new post.md”
  1. Open “new post”. Start writing by using a markdown editor (recommended).

#2 Preview

hugo server -D
  • Make sure everything works fine. Change draft:true to draft:false in the metadata section of the post.

#3 Git update

  1. Run Hugo.
hugo
  1. Go to “public” folder, push to GitHub. cd public.
git add .
git commit -m “xxxxx”
(git pull)
git push
  1. Go to the blog url to see the final result.

END.
