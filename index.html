Switch Between Multiple GitHub Accounts With SSH Keys

The following are some commands I use frequently when I switch between different GitHub accounts on my Mac. Use the following command to set a default Git account: Git config –global user.name "username1" Git config –global user.email "useremail1" Use the following command to switch to another Git account temporarily. Git config user.name "username1" Git config user.email "useremail1" Use the following command to check what is the current Git account: ssh -T git@github.
